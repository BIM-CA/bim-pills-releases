# BIM Pills

Herramientas inteligentes para optimizar tu flujo de trabajo en Autodesk Revit.

## Descarga

Descarga el instalador desde la sección [Releases](../../releases/latest) de este repositorio.

**Versión actual: beta 3.3**

Requiere Autodesk Revit 2024, 2025, 2026 o 2027.

## Instalación

1. Descarga **BIM Pills beta X.X Setup.exe** desde [Releases](../../releases/latest)
2. Ejecuta el instalador como administrador
3. Selecciona las versiones de Revit que tienes instaladas
4. Abre Revit — BIM Pills aparecerá en la cinta de opciones

## Actualizaciones

El plugin detecta automáticamente nuevas versiones al arrancar Revit y te ofrece descargar e instalar la actualización.
